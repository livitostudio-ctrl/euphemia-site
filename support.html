Questions fréquentes

L'application fonctionne-t-elle sans Internet ?

Oui. Euphemia est conçue pour fonctionner entièrement hors connexion.

J'ai perdu ma progression.

Les données sont enregistrées uniquement sur votre appareil. Si vous supprimez les données de l'application ou la désinstallez, votre progression est définitivement perdue.

Comment signaler une erreur dans un exercice ?

Envoyez-nous un e-mail en précisant :

  • le niveau concerné ;
  • le thème ;
  • si possible une capture d'écran.

Puis-je proposer une nouvelle fonctionnalité ?

Bien sûr ! Toutes les suggestions sont les bienvenues et nous aideront à faire évoluer Euphemia.
