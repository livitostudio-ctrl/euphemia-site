Assistance Euphemia

Vous rencontrez un problème, vous avez repéré une erreur dans un exercice ou vous souhaitez proposer une amélioration ? Livito Studio est à votre écoute.

Contacter l’assistance
Envoyez votre message à : livito.studio@gmail.com

Signaler un problème technique

Pour nous aider à comprendre et à résoudre rapidement le problème, indiquez si possible :

  • le modèle de votre téléphone ou de votre tablette ;
  • la version d’Android ou d’iOS utilisée ;
  • la version d’Euphemia installée ;
  • les étapes qui ont précédé le problème ;
  • une capture d’écran, si elle ne contient aucune information sensible.

Signaler une erreur dans un exercice

Malgré le soin apporté à la création et à la vérification des contenus, une erreur peut subsister.

Dans votre message, précisez :

  • le niveau scolaire concerné ;
  • le thème ou le type d’automatisme ;
  • la question affichée ;
  • la réponse proposée par l’application ;
  • une capture d’écran, lorsque cela est possible.

Proposer une amélioration

Les remarques des élèves, des parents et des enseignants contribuent directement à l’évolution d’Euphemia.

Vous pouvez nous écrire pour suggérer :

  • une nouvelle fonctionnalité ;
  • une amélioration de l’interface ;
  • un nouveau type d’exercice ;
  • une modification facilitant l’utilisation de l’application.

Questions fréquentes

L’application nécessite-t-elle une connexion Internet ?

Les fonctionnalités principales d’Euphemia sont accessibles sans connexion Internet.

Une connexion est uniquement nécessaire pour télécharger ou mettre à jour l’application, ainsi que pour ouvrir d’éventuelles ressources pédagogiques externes.

Faut-il créer un compte ?

Non. Euphemia ne nécessite aucun compte en ligne. Les profils sont créés et enregistrés uniquement sur l’appareil.

Où sont enregistrés les profils et les résultats ?

Les profils, les résultats, la progression et l’historique des erreurs sont conservés uniquement dans l’espace local de l’application sur l’appareil.

Pourquoi ai-je perdu ma progression ?

Les données n’étant pas synchronisées en ligne, elles peuvent être définitivement supprimées lorsque :

  • l’application est désinstallée ;
  • les données de l’application sont effacées dans les réglages ;
  • l’appareil est réinitialisé ou remplacé.

Comment supprimer mes données ?

Vous pouvez réinitialiser ou supprimer un profil directement depuis l’application.

Vous pouvez également effacer toutes les données d’Euphemia depuis les réglages de votre appareil ou en désinstallant l’application.

Euphemia contient-elle de la publicité ?

Non. La version actuelle d’Euphemia ne contient aucune publicité et ne collecte aucune donnée à des fins publicitaires.
